# Titanic Data Analysis

## Overview
This project analyzes the Titanic dataset using Python and visualizes survival patterns.

## Technologies Used
* Python
* Pandas
* Matplotlib
* Google Colab

## Project Description
The dataset contains passenger information such as age, sex, ticket class, fare, and survival status. The notebook performs data analysis and visualization to understand survival patterns.

## How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Run all cells to reproduce the analysis.
